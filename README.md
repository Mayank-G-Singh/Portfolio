# Personal Portfolio Website

This is my personal portfolio website, built to showcase my projects, skills, and experience as a first-year Computing student at TU Dublin. The website highlights my work in HTML, CSS and JS, and provides links to my GitHub repositories for my projects.



---

## Features

- About Me section with personal introduction and education
- College section enlists my skills I am developing
- Projects section showcasing my projects from Github
- Contact section with social links
- Responsive layout for desktop and mobile devices

---

## Technologies Used

- HTML5
- CSS
- Bootstrap
- JavaScript
- GitHub Pages for hosting
